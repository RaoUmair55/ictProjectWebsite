# Electric Bicycle Sharing and Sales Platform

Welcome to our repository for the electric bicycle sharing and sales platform. Here, you can explore and buy electric bicycles, as well as access our rental services.

## Exciting Features
- Easy browsing and purchasing of electric bicycles
- Convenient electric bicycle rental options
- User authentication and personalized profiles

## Installation Guide
1. Get started by cloning the repository
2. Then, simply install the necessary dependencies using `npm install`

- HTML
- CSS
